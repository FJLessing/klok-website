# Klok Webwerf

'n Eenvoudige webklok wat die huidige tyd in Afrikaans wys.

## Hoe dit werk
- **PHP** stel die aanvanklike tyd (index.php:16,29)
- **JavaScript** werk die tyd elke sekonde op (script.js:34)
- Tyd word in Afrikaanse woorde vertoon (script.js:26)
- **HTML fallback** vir bedieners sonder PHP-ondersteuning (index.html:18)

## Gebruik
- Met PHP: Laai op 'n PHP-bediener en besoek index.php
- Sonder PHP: Gebruik index.html as alternatief
